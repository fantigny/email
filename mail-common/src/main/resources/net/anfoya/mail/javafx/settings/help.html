in theory

  • a section contains labels
  • a thread is made of messages
  • threads are meant to be sorted with labels

drag to drop

  • drag down your thread to reply, forward, archive
  • drag a thread on the left to sort
  • drag a tag to customize
  • etc

press tab

  • once to search mails
  • again to search labels

reveal

  • find a hidden label in settings
  • show it again with a click

blue

  • is a color for system labels
  • can't move them
  • can't rename them
  • can't delete them
  • pfff

unread me

  • by dragging the thread to Unread label

be advanced

  • check a label to keep it in the selection
  • check another label to reduce the thread set

double click

  • in the thread set to reply a thread
  • reply all to be activated in settings

be extreme

  • show exclude box in settings and restart
  • include and exclude labels as you wish
  • deactivate archive on drop from settings
  • hide toolbar if you dare
  • cruise extreme mode
